# EGOS Self — Coding Preferences

## Python (CLI)
- Python 3.11+, type hints everywhere
- Click for CLI, Rich for output
- SQLite for local storage (append-only log)
- No external services required for core features
- Max 400 lines per module

## Kotlin (Android)
- Kotlin 2.0+, Jetpack Compose
- Material3 Dark theme (slate-900 background)
- Coroutines for async, StateFlow for UI state
- Max 500 lines per file

## General
- Conventional commits (feat:/fix:/chore:)
- No hardcoded secrets
- Offline-first architecture
- KDE Connect protocol v7 compatible
